# android-ProgressDialog
API Level 26からProgressDialogがdeprecatedに。

しかし俺は永遠に使うぞ

## Usage

* build.gradle

```
repositories {
    maven {
        url 'https://sugtao4423.github.io/android-ProgressDialog/repository'
    }
}

dependencies {
    implementation 'sugtao4423.support:progressdialog:1.0.0'
}
```
